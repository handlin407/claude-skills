# Test Files

This directory contains comprehensive test suites for each CRE database skill.

## Test Suites

- **transaction-comps-TEST-SUITE.md** - 12 test cases for property sales transactions
- **lease-comps-TEST-SUITE.md** - 12 test cases for tenant lease agreements
- **tenant-sales-TEST-SUITE.md** - 12 test cases for retailer revenue data

Each test suite includes:
- Complete test scenarios with expected behaviors
- Pre-test and post-test checklists
- Success criteria validation
- Cleanup protocols using `[TEST]` markers

## Usage

Test files are for development and QA purposes. They are NOT included in production `.skill` packages to keep package sizes optimized.
